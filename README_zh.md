# 知识图谱可视化工具

## 项目简介

这是一个基于Web的知识图谱可视化工具，允许用户通过简单的界面操作来展示和交互式探索知识图谱数据。

## 快速开始

1. 下载项目代码
2. 运行 `app.py` 启动应用
3. 应用将在本地端口5000上运行

## 功能特性

- **数据加载方式**：
  - 从预设文件中选择知识图谱数据
  - 上传本地JSON文件
- **可视化交互**：
  - 支持知识图谱的展示
  - 支持滑动、拖拽等操作
  - 点击节点可查看详细信息（显示在左下角）

## 数据格式要求

如需上传本地JSON文件，请确保符合以下格式：

```
{
  "nodes": [
    {
      "id": "节点唯一标识",
      "label": "节点标签",
      "properties": {
        // 可选，节点属性键值对
      }
    }
    // 更多节点...
  ],
  "relationships": [
    {
      "source": "起始节点ID",
      "target": "目标节点ID",
      "type": "关系类型"
    }
    // 更多关系...
  ]
}
```

## 使用说明

1. 启动应用后，在界面中选择：
   - 预设文件，或
   - 上传本地JSON文件
2. 点击"加载"按钮展示知识图谱
3. 在可视化界面中：
   - 使用鼠标拖拽移动图谱
   - 滚动鼠标滚轮缩放图谱
   - 点击节点查看详细信息

## 注意事项

- 确保上传的JSON文件格式正确
- 节点ID在relationships中的source和target字段必须与nodes中的id对应
